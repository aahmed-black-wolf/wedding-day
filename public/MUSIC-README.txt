Background music
================

A romantic track is already included: music.mp3
  -> Pachelbel's Canon in D (public-domain composition),
     recording from the Internet Archive (archive.org).

It starts when a visitor taps "افتح الدعوة" (open invitation) on the intro
screen and loops while they browse. A floating music button (bottom-left)
mutes/unmutes.

To use your OWN song instead, just replace this file:

    public/music.mp3

Notes:
- Browsers block silent autoplay, so the music intentionally starts on that
  first tap — this is expected behaviour, not a bug.
- If music.mp3 is ever missing, the site falls back to a built-in synthesized
  Canon in D melody so there is always music.
- Use a track you have the rights to if you replace it.
